# Typing Test

A modern, minimalist typing test application inspired by Monkeytype. Built with React 19, Vite, and Tailwind CSS 4.

![Typing Test Hero](src/assets/hero.png)

## 🚀 Features

- **Minimalist Design**: Clean and focused typing experience.
- **Multiple Themes**: Choose from Serika Dark, Carbon, Dracula, and Nord.
- **Word Categories**: Practice with general English, coding terms, or frequent words.
- **Detailed Statistics**:
  - WPM (Words Per Minute)
  - Raw WPM
  - Accuracy Percentage
  - Correct/Incorrect character counts
- **Performance Analysis**:
  - Real-time WPM chart during results.
  - Interactive Key Heatmap to identify weak spots.
- **Sound Effects**: Audio feedback for correct and incorrect keystrokes.
- **Keyboard Shortcuts**: Press `Tab` to quickly restart a test.

## 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Charts**: [Recharts](https://recharts.org/)
- **Testing**: [Vitest](https://vitest.dev/)

## 📦 Getting Started

### Prerequisites

- Node.js (Latest LTS recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd typing-test
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## 🧪 Testing

Run unit tests with Vitest:

```bash
npm run test
```
